## ⚙️ Pré-requisitos e Dependências
O projeto foi escrito em Python. Para executá-lo, você precisará ter o Python 3.14.3 instalado e as seguintes bibliotecas:

* `WebdriverManager`: Para utomação de navegadores web (como criar testes automatizados ou bots para extrair dados de sites).
* `Seenium`: Para utomação de navegadores web (como criar testes automatizados ou bots para extrair dados de sites).
* `Pandas` (`PIL`): Para análise e manipulação de dados (trabalha com tabelas de dados chamadas DataFrames).
* `Openpyxsl` (`PIL`): Para ler, escrever e modificar planilhas do Excel (`.xlsx`).

Você pode instalar todas as dependências rodando o seguinte comando:

```bash
 python -m pip install webdriver-manager
``` 

```bash
 python -m pip install selenium
```  

```bash
python -m pip install pandas
```  

```bash
 python -m pip install openpyxl
```
